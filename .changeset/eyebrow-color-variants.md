Add `purple` and `blue` color variants to `Eyebrow`

`Eyebrow` now accepts `color="purple"` and `color="blue"` in addition to `"red"`, matching the brand colors already available on `IconCircle`. Shades (`red-600`, `purple-800`, `blue-600`) match `IconCircle`'s background so eyebrow + icon read as a set when both appear.
